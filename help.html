Вариант №11

Язык - "Начало" Слагаемое ";" ... Слагаемое Опер ... Опер "Окончание"
Слагаемое = "Анализ" Перем ! "Синтез" Перем ";" ... Перем
Опер = Перем "=" Прав.часть

В правой части - переменные и вещественные числа, соединены аддитивными, мультипликативными и логическими операциями; есть круглые скобки с любой глубиной вложенности.
Переменная - первая буква, затем до трех цифр.. Русский алфавит, десятичная арифметика.

BNF

Язык = "Начало" Слагаемое ";" ... Слагаемое Опер ... Опер "Окончание"
Слагаемое = "Анализ" Перем ! "Синтез" Перем "," ... Перем
Опер = Перем "=" Прав.часть
Прав. часть = Блок1 Знак1 ... Блок1
Знак1 = "+" ! "-"
Блок1 = Блок2 Знак2 ... Блок2
Знак2 = "*" ! "/"
Блок3 = Блок4 Знак3 ... Блок4
Знак3 = "И" ! "ИЛИ"
Блок4 = Блок5
Блок5 = Перем ! Вещ ! "(" Прав.ч. ")"
Перем = Бук
Бук = "А" ! "Б" ! ... ! "Я"
Вещ = Цел "." Цел
Цел = Циф ... Циф
Циф = "0" ! "1" ! ... ! "9"

Свойства языка

Приоретет операторов в порядке убывания:

1 - "НЕ"
2 - "И", "ИЛИ"
3 - "*", "/"
4 - "+", "-"
5 - "-" (унарный минус) Выражения в скобках имеют наивысший приоретет (обычно в ЯП приоретет логических операций выше).

Ассоциативность

Левоассоциативные бинарные операторы: "И", "ИЛИ", "*", "/", "+", "-"
Правоассоциативные унарные операторы: "НЕ", "-"

Логические операции:

0.0 -- FALSE, любое число отличное от нуля -- TRUE
Операторы "НЕ", "И", "ИЛИ" возвращают 1.0 если аргумент(ы) вычисляются в TRUE
Оба оператора ленивые. Т.е. если бы скобки имели приоретет ниже чем И ИЛИ, то было бы так:
0.0 И (1.0 / 0.0) -> 0.0
1.0 И (1.0 / 0.0) -> Ошибка: деление на ноль!
1.0 ИЛИ (1.0 / 0.0) -> 1.0
0.0 ИЛИ (1.0 / 0.0) -> Ошибка: деление на ноль!

Регистр

Регистр символов имеет значение.

Унарный минус

Для использования унарного минуса в выражениях, он должен находиться в самой левой части выражения.
Скобки представляют собой отдельные выражения.
Например: - 1 - (- 1 - 1) + (- 1) -- правильно. 5 - - 5 -- ошибка.

Примеры

Программа №1

Начало
Синтез а, б, б1
а = 1.0 + 1.0 + 1.0
б = а * 5.0
б = б - 3.0
Окончание

Вывод:
а = 3.0
б = 12.0

Программа №2

Начало
Анализ а112;
Анализ б911;
Анализ в
в = 0.0 / 0.0 + 0.0
Окончание

Вывод:
Ошибка: деление на ноль!

Программа №3

Начало
Синтез а, б, в;
Синтез а1, б1, в1
а1 = 0.0 ИЛИ 5.0 + 1.0
Окончание

Вывод:
а1 = 2.0

Программа №4

Начало
Синтез П1
П1 = П1
Окончание

Вывод:
Ошибка: перед использованием переменной, ей должно быть присвоено значение!
